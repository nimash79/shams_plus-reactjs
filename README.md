# Shams Plus - reactjs

Shams Plus bank
This is front-end of this project.
You can see back-end from [here](https://github.com/nimash79/shams_plus-nodejs)

## Installation

Use [npm](https://npmjs.com/) to install packages.

```bash
npm install
```

## Usage

```bash
npm start
```
